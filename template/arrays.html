    An array is a fixed-size data structure that stores elements of the same data type in contiguous memory locations. On this page it shows you how to add, delete, and find names in an array, giving a clear visual of how these actions impact the data structure. You add a name using the “Add Name” button. After you add a name it will show up in “Names in a Table”. To delete that name you just added, you have to press on the index number next to the name and then click okay. To find a name just enter the name and hit find.
